Model Context Protocol (MCP)

Ein aufstrebender offener Standard, der es KI-Agenten ermöglicht, externe Tools und andere Agenten sicher zu entdecken, zu verstehen und zu nutzen — der „USB-C der Agenten-Interoperabilität“, der im Multi-Agenten-Ökosystem rasch an Fahrt gewinnt.

Was ermöglicht MCP eigentlich?

  • Entdeckt verfügbare Tools automatisch (keine hartkodierten Integrationen)
  • Versteht, was jedes Tool tut, seine Parameter und erforderlichen Berechtigungen
  • Ruft jedes MCP-kompatible Tool auf (Web-Suche, Dateisystem, GitHub, Datenbanken, Rechner, Wikis, APIs…)
  • Funktioniert framework-übergreifend — LangChain, CrewAI, Autogen, LlamaIndex, Semantic Kernel und viele andere unterstützen oder planen bereits MCP
  • Unterstützt Human-in-the-Loop — sensible Aktionen können eine explizite Nutzerfreigabe erfordern, bevor sie ausgeführt werden
  • Ermöglicht Multi-Agenten-Kollaboration via A2A (Agent-to-Agent) über dasselbe Protokoll

Typische Tools, die heute über MCP verfügbar sind (2025/2026)

  • URL abrufen / Web-Browser-Tools
  • Dateisystem-Lese-/Schreiboperationen
  • GitHub (Issues, PRs, Repo-Zugriff)
  • DeepWiki / Suche in interner Wissensbasis
  • Zeit- & Datumsummwandlungen, mathematische Rechner
  • Jeder REST/GraphQL-Service, der als MCP-Tool umgesetzt wurde
  • Datenbank-Abfrage-Tools (SQL, Vektor-DBs)

Warum Camunda 8 + MCP die perfekte Enterprise-Kombination ist

  • 100% Sichtbarkeit & Nachverfolgbarkeit — jeder MCP-Tool-Aufruf ist eine reguläre BPMN-Service-Task
  • Eingebaute Guardrails — DMN-Tabellen entscheiden, welche Tool-Aufrufe eine Nutzerbestätigung benötigen
  • Keine Black-Box-Agenten — vollständige Nachverfolgung dessen, was der Agent gesehen, entschieden und ausgeführt hat
  • Human-in-the-Loop an jeder Stelle — Eskalation zur Tasklist mit einem Klick
  • Agenten mischen und kombinieren — Camunda-Agenten neben OpenAI, Anthropic, Mistral oder jedem anderen MCP-kompatiblen Agenten einsetzen
  • Skaliert auf Millionen von Tool-Aufrufen pro Tag auf der cloud-nativen Zeebe-Engine

Fazit: MCP ist nicht nur ein weiteres Integrationsformat — es ist der universelle Stecker, der es Unternehmen endlich ermöglicht, KI-Agenten sicher und governabel echte Macht zu geben, indem sie sie mit Hunderten von Tools verbindet, während alles transparent, nachverfolgbar und unter voller Prozess-Orchestrierungs-Kontrolle mit Camunda bleibt.

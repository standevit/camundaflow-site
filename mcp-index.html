Model Context Protocol (MCP)

An emerging open standard that lets AI agents securely discover, understand, and use external tools and other agents — the “USB-C of agent interoperability” that is rapidly gaining traction in the multi-agent ecosystem.

What does MCP actually enable?

  • Discovers available tools automatically (no hard-coded integrations)
  • Understands what each tool does, its parameters and required permissions
  • Calls any MCP-compatible tool (web search, filesystem, GitHub, databases, calculators, wikis, APIs…)
  • Works across frameworks — LangChain, CrewAI, Autogen, LlamaIndex, Semantic Kernel and many others already support or plan MCP
  • Supports human-in-the-loop — sensitive actions can require explicit user approval before execution
  • Enables multi-agent collaboration via A2A (Agent-to-Agent) over the same protocol

Typical tools available today via MCP (2025/2026)

  • Fetch URL / Web Browser tools
  • Filesystem read/write operations
  • GitHub (issues, PRs, repo access)
  • DeepWiki / internal knowledge base search
  • Time & date conversions, math calculators
  • Any REST/GraphQL service wrapped as an MCP tool
  • Database query tools (SQL, vector DBs)

Why Camunda 8 + MCP is the perfect enterprise combination

  • 100% visibility & auditability — every MCP tool call is a regular BPMN service task
  • Built-in guardrails — DMN tables decide which tool calls need user confirmation
  • No black-box agents — full trace of what the agent saw, decided and executed
  • Human-in-the-loop at any point — escalation to Tasklist with one click
  • Mix & match agents — use Camunda agents alongside OpenAI, Anthropic, Mistral or any MCP-compatible agent
  • Scales to millions of tool calls per day on Zeebe cloud-native engine

Conclusion: MCP is not just another integration format — it is the universal plug that finally lets enterprises safely and governably give AI agents real power by connecting them to hundreds of tools, all while keeping everything transparent, auditable and under full process-orchestration control with Camunda.
